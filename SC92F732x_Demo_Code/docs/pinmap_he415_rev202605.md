## MCU 型号

## P2.6 双灯三态
`P2.6` 是反向并联双灯结构，并且有上下拉电阻。软件不能再把它当作普通低亮 LED。

控制关系：
1. `P2.6=0`：点亮 D4
2. `P2.6=1`：点亮 D5
3. `P2.6` 配成输入/高阻：D4、D5 都灭

建议软件接口：
1. `LED_WATER_SMALL_ON`
2. `LED_WATER_STOP_ON`
3. `LED_WATER_BOTH_OFF`

具体 D4/D5 分别对应“小水量灯/停止灯”的面壳含义，建议上板再按丝印或实物灯位复核一次。

## 代码改动要点
1. `IN_K3_PIN` 已从 `P03` 改为 `P25`。
2. `P25` 已配置为输入上拉，不再作为 LED 输出。
3. `P03/P02` 已作为模拟 I2C 引脚，分别接 24C02 `SCL/SDA`。
4. `P2.6` 已封装为三态控制，不再沿用普通 `LED_MODE2_ON/OFF` 语义。
